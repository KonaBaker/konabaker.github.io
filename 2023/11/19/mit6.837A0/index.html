MIT 6.837 assignment 0

这次作业主要是用于熟悉C++的相关使用。
作业题目：6.837 Assignment 0 (mit.edu)

作业简述

使用C++和相关的图片(image.h)和矩阵(matrix.h)库实现一个分形图形的绘制。作业中要求实现一个IFS类，封装相关操作，自己编写头文件和实现文件。这个其实仅以分形为背景，重点在实现，作业中已经给出伪代码，只关注实现伪代码即可，不用过多关注分形原理。

想了解迭代分形原理的可以看作业中给出的两个链接：
1.* spanky
2.* ifs

实现伪代码：

主要过程就是随机生成一些点，然后经过iters次迭代变换后(依概率选择变换矩阵)，在image的像素上着色就可以了。

代码

ifs.h

ifs.cpp::readData()

这里我没有使用作业中命令行读文件的方式，我是直接读入相关数据，在这里我重构了一下matrix中的read3x3方法。

ifs.cpp::render(Image& image, int points, int iters)

这里读入的三个参数是，一个作为输出数据的图像对象image(封装在image.h中)，points是渲染的点数，iters是迭代次数。这里直接就按上面的伪代码实现就可以了。

利用rand()函数生成随机的点，然后按给出的概率选择，这里选择部分使用的是区间法，基本思想就是，生成一个随机数，判断随机数落在那个区间内就选择哪个矩阵，这个区间是按照存储顺序，从零开始按照概率累加的。

不太明白的可以看力扣上的一道题，就是实现这个算法：LCR 071. 按权重随机选择 - 力扣（LeetCode）

最后在main函数中创建对象运行就可以了。

tips&P.S.

1.作业中给的.C文件可能会让IDE自动使用C的编译器，可能会识别不了C++语法，class等关键字可能会报错，全部使用.cpp就可以了。
2.fopen安全性报错，在项目->属性->C++->预处理器中添加 _CRT_SECURE_NO_WARNINGS 即可。
3.这里还有一个就是生成随机坐标一定要归一化后再转换到窗口上(我觉得可能是精度问题？我也没搞太清楚)。
4.文件读写和命令行的方式可以看参考链接中其他作者的。
